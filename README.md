#  Pizza Order Project

## Overview

A simple desktop application that allows users to build their own pizza and get instant order details.

---

## ✨ Features

- Choose pizza size
- Select crust type
- Add multiple toppings
- Choose between Eat In or Take Out
- View live order summary
- Calculate total price dynamically

---

## 🧩 UI Components Used

- **RadioButton** → Used to select:
  - Pizza size
  - Crust type
  - Eat In / Take Out

- **CheckBox** → Used to select toppings

- **Labels** → Used to display the order summary

- **Buttons**:
  - **Order** → Confirm the order
  - **Reset** → Reset all selections

---

## ⚡ How It Works

- Every user action updates the order summary instantly
- The total price is calculated dynamically based on selections
- The `Tag` property is used to store item prices and simplify calculations
